【每日一项目】Manifest - OpenClaw 智能 LLM 路由插件

发表于 2026-03-22 分类于 每日一项目

项目概览

Manifest 是一个 OpenClaw 插件，通过 23 维评分算法在 <2ms 内将用户请求路由到最合适的大模型，实测节省成本最高可达 70%。

很多场景下我们用 GPT-4o 或 Claude Opus 处理一个”今天天气怎么样”这类简单查询，Manifest 解决的就是这个资源浪费问题。

  • GitHub: mnfst/manifest
  • 开源协议: MIT
  • 语言: TypeScript
  • 支持模型: 300+（OpenAI、Anthropic、Google、DeepSeek、xAI、MiniMax 等）

2021，我裸辞了

发表于 2021-08-07 更新于 2026-03-20 分类于 随笔

2021年7月23日，我结束了在公司的最后一天。这是我工作以来第一次裸辞。

last day 最后一天并没有想象中那么轻松愉悦，因为我还没有找到工作。IT同事提醒我今天早点去把电脑等设备还了。心里略微有些感伤。电脑还了之后，收拾完东西就下楼了。就像请假了一样，没有跟同学们说再见，没有太多留恋。去一楼还工牌，前台小姐姐竟然当着我的面把工牌扔进了垃圾桶。

愉快的假期当然还是拿到了 offer。入职前开启了一段愉快的假期——疫情使节约了不少钱，只能上海本地游。可能是裸辞后最开心的一段时间，没有找工作压力，尽情放纵。找工作时竟然瘦了 6 斤，各种美食补回来。

写在最后：第一次裸辞，算是平稳收场了。未来充满不确定性，还是要坚定自己的规划与打算。感谢老婆一直以来的支持和鼓励。
