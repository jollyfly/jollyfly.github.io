项目描述： Interactive roadmaps, guides and other educational content to help developers grow in their careers.

核心特性

  1. 交互式学习路线图：提供前端/后端/DevOps/AI 等多条职业发展路径的视觉化路线图
  2. 年度版本更新：每年更新一次，反映行业最新技术趋势
  3. 节点详情：每个知识点节点都有对应的文章或资源链接
  4. 社区共建：任何人都可以通过 PR 提交改进建议
  5. 多格式输出：支持 SVG 矢量图、PDF、在线互动等多种格式

技术架构深度剖析

一、技术栈

技术选型：

组件 技术 选型理由
静态网站框架 Astro 零 JS 默认、构建快速、SSR/SSG 灵活
端测试 Playwright 支持多浏览器、跨平台
数据存储 JSON 无需数据库，Git 管理版本
图标 SVG 矢量可缩放，颜色可控
部署 Vercel/Netlify 支持 Astro 一键部署

二、知识图谱数据模型

文件：public/roadmaps/ 下的 JSON 文件

路线图数据以 JSON 格式存储，可以被任何前端框架解析：

数据驱动渲染的优势：

  1. 同一数据多端使用：同一 JSON 可以渲染网页、移动端、PDF
  2. 版本管理：Git 天然记录了路线图的版本历史
  3. 机器可处理：可以写脚本分析各知识点的关联
  4. 协作友好：JSON diff 比图片 diff 更精确

三、内容生成脚本

文件：scripts/editor-roadmap-content.ts

脚本的职责：

  • 将 JSON 数据渲染成文章页面
  • 从外部 API 获取相关资源（MDN 链接、教程推荐）
  • 生成每个知识点的 SEO meta 信息
