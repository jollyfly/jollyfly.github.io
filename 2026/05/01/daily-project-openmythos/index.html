项目描述： A theoretical reconstruction of the Claude Mythos architecture, built from first principles using the available research literature. 基于公开研究与推测，对 Claude Mythos 模型架构的理论复现实现。

核心特性

Recurrent-Depth Transformer（RDT）架构

OpenMythos 假设 Claude Mythos 采用的是循环深度 Transformer（Recurrent-Depth Transformer），即 Looped Transformer。核心思想是：并非堆叠数百个独立层，而是让一部分层在单次前向传播中循环使用多次。相同权重，更多循环，更深思考。

三阶段架构

可切换注意力机制

支持 MLA（Multi-head Latent Attention） 和 GQA（Grouped Query Attention） 两种注意力模式，通过 attn_type 参数指定。

Sparse MoE（混合专家）前馈网络

使用路由专家（routed experts）和共享专家（shared experts），适合探索计算自适应、深度可变推理场景。

光谱半径稳定性验证

通过 torch.linalg.eigvals(A).abs().max() 计算矩阵 A 的特征值最大模长 ρ(A)，用于验证循环隐状态是否稳定收敛（必须 < 1）。

训练基础设施

  • 数据集：FineWeb-Edu（HuggingFaceFW/fineweb-edu）
  • Tokenizer：openai/gpt-oss-20b via MythosTokenizer
  • 并行训练：PyTorch DDP + torchrun
  • 精度：H100/A100 用 bfloat16，老卡用 float16 + GradScaler
  • 调度：2000 步 warmup → cosine decay
  • 目标：30B tokens（Chinchilla 风格调整）

关键创新点

1. 循环推理 vs CoT（Chain-of-Thought）

传统 CoT 在解码过程中输出中间 token，而 RDT 在单次前向传播内，于连续隐空间中完成所有推理，无中间 token 输出，效率更高。

2. 计算自适应深度

通过 max_loop_iters 控制循环次数，不同输入可使用不同推理深度，实现计算资源的自适应分配。

3. MLA 低秩注意力

相比标准 MHA，MLA 通过低秩分解显著降低 KV 缓存开销，适合长上下文场景。

总结

OpenMythos 是一个极具研究价值的 LLM 架构实验场，通过开源方式验证了”循环 transformer”这一新兴架构方向的可能性。其设计理念（单次前向传播内的隐空间推理）对未来高效推理模型有重要参考意义。
