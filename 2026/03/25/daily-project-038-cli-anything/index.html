项目概览

CLI-Anything（HKUDS/CLI-Anything）是香港大学数据系统实验室（HKUDS）开源的 AI Agent 工具桥接框架。核心理念：任何有 GUI 的桌面软件，都可以一键生成一个完整的 CLI 接口，让 AI Agent 像操作代码一样操作任何软件。

⭐ 22,588 GitHub Stars | 11+ 软件支持 | Claude Code/OpenClaw/Codex 全支持 | 7 阶段自动生成

支持的软件包括 GIMP、Blender、LibreOffice、MuseScore、Shotcut、Zoom 等，覆盖图像编辑、3D 建模、办公套件、音频视频编辑、视频会议等场景。而且这个列表还在以每天更新的速度扩张。

核心特性

1. 七阶段自动 CLI 生成管道

整个流程一条命令，AI 自动完成从源码分析到可分发包的全流程。

2. CLI-Hub：AI Agent 的应用商店

CLI-Hub 的元数据自动从 registry.json 生成（通过 GitHub Actions），每次 PR 合并后 Hub 即时更新。整个生态是 agent-native 的，意味着 AI Agent 可以自主发现和安装新的 CLI。

3. Agent 技能发现（SKILL.md Generation）

Phase 6.5 新增的能力：每个生成的 CLI 都自带 SKILL.md，AI Agent 安装后无需查看文档，直接问 /cli-anything:cli-anything ./gimp --help 就能理解所有可用命令。

4. 结构化输出设计

所有 CLI 命令都支持 --json 输出模式：

JSON 输出使 Agent 能够确定性解析，而不是靠正则表达式从 stdout 里猜内容。

5. 撤销/重做支持

这对 Agent 尤其重要：AI 执行操作前可以先 dry-run 试错，发现问题立即回滚，不污染最终输出。

适用场景

非常适合的场景

  1. AI Agent 操作专业桌面软件

    • Claude Code 想自动化 GIMP 做产品图片处理？
    • 想让 Agent 自动生成 Blender 3D 渲染图？
    • CLI-Anything 就是这个桥梁

  2. 批量自动化重复性桌面操作

    • “每天把 100 张图片统一调色 + 裁剪 + 导出”
    • LibreOffice Writer 批量文档格式转换
    • Shotcut 批量视频转码

  3. 软件测试自动化

    • 每个生成的 CLI 都自带完整测试计划
    • 可集成到 CI/CD 流水线中

  4. 构建 agent-native 软件生态

    • 开发者为自己的软件写一个 harness → 自动获得 Claude Code 插件
    • CLI-Hub 的 agent-native 分发机制让安装无缝

不适合的场景

  • 没有源代码的软件（无法分析 → 只能基于公开 API 的黑盒封装）
  • 实时交互密集型软件（游戏、实时音视频编辑器）
  • 需要图形反馈的手动操作（CLI 天然不适合高度视觉化的交互）

快速上手

总结

CLI-Anything 的核心洞察是：CLI 是 AI Agent 的母语。Text commands match LLM format，结构化输出消除了解析复杂度，undo/redo 支持让 AI 敢于试错。GUI 软件虽然人类友好，但对 Agent 来说是一堵黑墙。CLI-Anything 在这堵墙上开了一扇精确的门，让 Agent 能够确定性、可审计地控制任何软件。

“Today’s Software Serves Humans👨💻. Tomorrow’s Users will be Agents🤖.”

CLI-Anything 是这个 transition 最务实的实现路径之一。
