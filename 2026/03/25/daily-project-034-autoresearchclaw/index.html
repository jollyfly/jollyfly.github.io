daily project | AutoResearchClaw 全自主研究流水线从想法到论文

项目概览

AutoResearchClaw（aiming-lab/AutoResearchClaw，⭐8,505）是一个全自主的 23 阶段研究流水线，能够将一个研究想法直接转化为会议级别的学术论文，全程无需人工干预。

核心流程：输入一个研究主题 → 自动完成文献调研（OpenAlex / Semantic Scholar / arXiv）→ 设计实验方案 → 编写实验代码 → 执行沙箱实验 → 统计分析结果 → 生成 LaTeX 论文 → 多 Agent 同行评审 → 输出可投递的最终稿件。

支持通过 OpenClaw 在 Discord、Telegram、飞书、微信等平台直接启动研究任务，真正实现”发一条消息，等一篇论文”。

核心特性

1. 23 阶段端到端流水线

AutoResearchClaw 将完整的研究流程分解为 23 个自动化阶段：

阶段范围 核心步骤
想法验证（1-3） 想法可行性评估、研究现状扫描、novelty 分析
文献调研（4-7） arXiv 论文检索、OpenAlex 引用分析、Semantic Scholar 关系图谱
方法设计（8-11） 核心算法设计、Baseline 选择、实验方案规划
代码实现（12-15） 实验代码编写（CodeAgent）、Baseline 实现（BenchmarkAgent）、图表生成（FigureAgent）
实验执行（16-19） GPU/MPS/CPU 自动检测、沙箱执行、结果收集
论文撰写（20-22） Introduction → Related Work → Method → Experiments → Results → Conclusion
质量审核（23） 4 层引用完整性验证、AI-Slop 检测、7 维评审评分、NeurIPS 检查清单

输出物详解

一次完整的 researchclaw run 产出以下文件（位于 artifacts/rc-{timestamp}-{hash}/deliverables/）：

文件 说明
paper_draft.md 完整学术论文（Intro → Related Work → Method → Experiments → Results → Conclusion）
paper.tex 会议级别 LaTeX（支持 NeurIPS/ICML/ICLR 模板）
references.bib 从 OpenAlex / Semantic Scholar / arXiv 抓取的 BibTeX，自动裁剪到正引用
verification_report.json 4 层引用完整性验证报告
reviews.md 多 Agent 同行评审（含 methodology-evidence 一致性检查）
experiment_runs/ 所有实验代码 + 沙箱执行结果 + JSON 格式指标
charts/ 自动生成的条件对比图（含误差棒和置信区间）
evolution/ 从本次运行中提取的自学习 lessons（供下次 run 使用）

适用场景

最适合

  1. 科研加速：快速验证一个想法是否值得深入，避免花 3 个月写完论文发现撞车了
  2. 论文初稿生成：作为第一稿基础，人工再加工比从零写快 5-10 倍
  3. 跨学科探索：对陌生领域快速建立文献全景图和基线认知
  4. 想法压力测试：用流水线自带的 4 层引用验证 + Slop 检测来评估想法的真实性

不适合

  • 需要突破性 novel 贡献的顶级会议论文（当前流水线生成内容偏向组合式创新）
  • 需要精确控制实验细节的研究（自愈系统可能产生非预期的实验配置变更）
  • 依赖专有数据集或私有代码的实验（沙箱隔离环境无法访问本地资源）

总结

AutoResearchClaw 是目前最完整的”想法 → 论文”全自动化开源实现。它的 23 阶段流水线 + 多 Agent 协作 + 防伪造引用验证 + 自愈实验循环，构成了一套真正可用的科研加速基础设施。

最值得关注的是它的 MetaClaw 跨 run 学习机制（v0.3.0+）：每次失败的实验都会提取为 structured lesson 并注入到后续 23 个阶段中，形成真正的”自我进化”能力。如果这一机制持续打磨，AutoResearchClaw 有望成为 AI for Science 领域的基础设施级项目。
